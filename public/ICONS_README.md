# PWA Icons Required

The following icon files need to be added to the `public/` directory:

- `icon-192x192.png` - 192x192 pixel icon for PWA
- `icon-512x512.png` - 512x512 pixel icon for PWA
- `icon.png` - Default icon for notifications
- `badge.png` - Badge icon for notifications

You can generate these icons using tools like:

- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

The icons should represent the MONEE brand and work well at different sizes.
