**Q: I marked something as paid, but don't see the expense**
A: Check your Expenses list - it's there! Filter by date or category.

**Q: Can I change the amount each time?**
A: Currently, it uses the set amount. Edit the recurring record if it changes permanently, or manually add an expense for one-off variations.

**Q: What if I miss a payment?**
A: Just mark it as paid whenever you do pay. The next due date adjusts from the current date.

**Q: Can I delete a recurring expense?**
A: Not yet, but you can pause it indefinitely. Delete feature coming soon!
